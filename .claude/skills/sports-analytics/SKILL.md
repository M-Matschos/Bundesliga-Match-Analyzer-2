# Sports-Analytics

Führe Sportanalytik für das Match Oracle Decision Intelligence System durch. Generiere Match-Vorhersagen, Spielerleistungs-Analysen, Team-Analysen, Expected Goals (xG) Berechnungen und Momentum-Scoring.

## Analyse-Module

1. **Match-Vorhersagen**
   - Ergebnis-Wahrscheinlichkeit (Home, Draw, Away)
   - Erwartete Tore (Over/Under Prognosen)
   - Goal-Differential Vorhersagen
   - Confidence-Scores per Modell
   - Multi-Model Ensemble

2. **Spielerleistungs-Analyse**
   - Erwartete Tore (xG) pro Spieler
   - Expected Assists (xA)
   - Passing Accuracy
   - Defensive Actions
   - Form-Trend (letzte 5/10 Spiele)
   - Injury-Status Berücksichtigung

3. **Team-Analyse**
   - Offensive/Defensive Stärke
   - Home/Away Performance-Differential
   - Formation-Tendenzen
   - Kopfball-Dominanz
   - Set Piece Effectiveness
   - Head-to-Head Historie

4. **xG-Berechnungen**
   - Shot-Quality-Bewertung (Position, Angle, Defender-Nähe)
   - Cumulative xG (pro Spieler, pro Team)
   - xG Underperformance/Overperformance
   - Trend-Analyse (Saison)

5. **Momentum-Scoring**
   - Aktuelle Form-Indikator (letzte 3/5/10 Spiele)
   - Gewinn-Streak vs Verlust-Streak
   - Zuhause-Boden-Effekt
   - Psychologische Faktoren (Derby, Aufstieg-Relevanz)

## Ausgabeformat

1. Match-Vorhersage-Report mit Wahrscheinlichkeiten
2. Spieler-Leistungs-Rankingslisten
3. Team-Vergleich-Matrix
4. xG-Visualisierungen (Feldposition-Heatmaps)
5. Momentum-Indikator-Dashboard
6. Vertrauens-Scores pro Modell
7. Odd-Vergleich mit Wahrscheinlichkeits-Abweichungen
