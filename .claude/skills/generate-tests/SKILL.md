# Generate-Tests

Generiere automatisierte Tests für das Match Oracle Decision Intelligence System. Unterstütze Jest Unit-Tests, React Native Testing Library Component-Tests und Detox E2E-Tests mit vollständiger Abdeckung.

## Test-Typen

1. **Unit-Tests** (Jest)
   - Funktions-Tests mit Mocks
   - Edge Cases und Error Paths
   - Snapshot Tests für Datenstrukturen
   - Performance-Tests

2. **Component-Tests** (React Native Testing Library)
   - Render-Tests
   - User Interaction Tests
   - Props-Variationen
   - Accessibility Tests (a11y)

3. **E2E-Tests** (Detox)
   - User-Flow Tests
   - Cross-Screen Navigation
   - Daten-Persistierung
   - Offline/Online-Szenarien

## Ausgabeformat

1. Generierte Test-Suite (Jest)
2. Generierte Component-Tests (RNTL)
3. Generierte E2E-Tests (Detox)
4. Test-Datei-Struktur (.test.ts, .e2e.ts)
5. Mocking-Strategie für Abhängigkeiten
6. Coverage-Ziele (80%+ angestrebt)
7. Test-Dokumentation mit Szenarien
