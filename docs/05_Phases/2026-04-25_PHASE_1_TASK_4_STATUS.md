## ⚠️ Known Issues & Mitigations

### Version Mismatch Warnings
**Issue:** Expo 55 recommends newer versions of some packages
- react-native: Current 0.73.0 → Recommended 0.83.6
- react: Current 18.3.1 → Recommended 19.2.0
- @react-native-async-storage: Current 1.24.0 → Recommended 2.2.0

**Impact:** Minor — warnings only, app builds and runs successfully  
**Mitigation:** Can be updated in Phase 2 if stability issues arise

### Port Conflict (Resolved)
**Issue:** Metro Bundler port 8081 was in use  
**Solution:** Use `npx expo start --web --port 8082`  
**Status:** ✅ Resolved

### Windows Node Process Management
**Issue:** Windows shell doesn't cleanly terminate node processes  
**Solution:** Use `pkill -f node` before starting new instances  
**Status:** ✅ Documented in BUILD_GUIDE.md

### EAS Authentication
**Issue:** EAS Cloud builds require login
**Solution:** 
- User must run: `npx eas login`
- Or set EXPO_TOKEN environment variable for CI/CD
**Status:** ✅ Documented with step-by-step guide

3. **"Dependency resolution"** → Reinstall: `rm -rf node_modules && npm install`
