# 🗺️ Entwicklungs-Roadmap

---

## Phase 1 — MVP (Monate 1–3)

**Ziel: Funktionsfähige App mit Bundesliga-Kern**

- [ ] Projektstruktur aufsetzen (dieses Repository)
- [ ] Datenbank mit Schema + Seeds aufsetzen
- [ ] football-data.org + API-Football Collector
- [ ] Poisson + Monte Carlo Modell (ohne ML)
- [ ] FastAPI Backend: `/matches`, `/predictions`, `/calculate/weekend`
- [ ] Weekend Calculator Button (Kernfeature)
- [ ] React Native App: Dashboard + Match-Detail
- [ ] Bundesliga 1+2 (10 Jahre Historik importieren)
- [ ] Virtuelles Wetten (Basis)
- [ ] App Store Submission

**Liefert:** Funktionierende App mit BL1+BL2 Weekend-Berechnung

## Phase 2 — Ausbau (Monate 4–6)

**Ziel: Premium-Features + weitere Ligen**

- [ ] XGBoost ML-Modell trainieren und deployen
- [ ] SHAP-Erklärungen im Match-Detail
- [ ] Alle 6 Ligen einbinden
- [ ] Spieler-Modul vollständig (Einfluss-Score)
- [ ] Tipico Affiliate Deep-Links
- [ ] OddsAPI: Quoten-Vergleich + Value-Bet-Detection
- [ ] Premium-Abo (In-App Purchase, 9,99€/Monat)
- [ ] Push-Notifications (Verletzungs-Alerts)
- [ ] Transfermarkt Scraper (Verletzungen live)

**Liefert:** Vollständiges Produkt, Monetarisierung aktiv

## Phase 3 — Wachstum (Monate 7–12)

- [ ] Leaderboard + Tipp-Gruppen (Social Features)
